# Dataset Schema

**Version:** 1.0
**Created:** Phase 1 — Data Foundation
**Format:** JSONL with fields `prompt` (string) and `price` (float)

## Record Format

Each JSONL record has exactly two fields:

```json
{"prompt": "Property: Single Family in zip 07650. 3 bedrooms, 2.0 bathrooms, 1500 sqft living area, 0.25 acre lot, built in 1990. Predicted price: $", "price": 450000.0}
```

The `prompt` field is generated by `lambda/prompt_utils.format_prompt()`. Do NOT construct prompts manually.

## Feature Fields (used to generate prompt)

| Field | Type | Source | Range | Notes |
|-------|------|--------|-------|-------|
| bedrooms | int | synthetic / SR1A | 1–6 | Integer only |
| bathrooms | float | synthetic / SR1A | 1.0–5.0 | Half-bath increments (1.0, 1.5, 2.0, ...) |
| sqft | int | synthetic / MODIV | 500–8000 | Living area only, not lot |
| lot_size | float | synthetic / MODIV | 0.05–10.0 | Acres; formatted to 2 decimal places in prompt |
| year_built | int | synthetic / MODIV | 1900–2024 | Calendar year |
| zip_code | str | synthetic / SR1A | NJ zip codes | 5-digit string; leading zero preserved (e.g., "07650") |
| property_type | str | synthetic / SR1A class code | Single Family, Condo, Townhouse, Multi-Family | SR1A class 2 = residential |

## Label and Metadata Fields

| Field | Type | Source | Range | Notes |
|-------|------|--------|-------|-------|
| price | float | SR1A sale price / synthetic | $100,000–$3,000,000 | Sale price (not assessed value); rounded to nearest $100 |
| source | str | metadata | "real" / "synthetic" | "real" = price from SR1A public data; "synthetic" = generated |
| county | str | derived from zip / SR1A county code | 21 NJ counties | Bergen, Morris, Essex, Hudson, etc. |

## Public Data Sources (DATA-04)

At least 30% of records have `source: "real"`. Real records derive from:

- **NJ Division of Taxation SR1A Sales File (2024)**
  URL: https://www.nj.gov/treasury/taxation/lpt/statdata.shtml
  Fields used: sale price, property class code, county code, municipality code
  Fields synthesized (if missing): bedrooms, bathrooms, sqft (filled from county log-normal distributions)

- **NJ MODIV Property Assessment Files (2024)**
  URL: https://www.nj.gov/treasury/taxation/lpt/statdata.shtml (same page)
  Fields used: living area (sqft), year built — joined to SR1A on parcel ID

## Splits

| File | Location | Size (target) | Ratio |
|------|----------|----------------|-------|
| train.jsonl | data/splits/train.jsonl | ~70% of total | 70% |
| val.jsonl | data/splits/val.jsonl | ~15% of total | 15% |
| test.jsonl | data/splits/test.jsonl | ~15% of total | 15% |

Total target: 5,000–10,000 records. Stratified by price quartile.

## Distribution Validation

Generated price distribution must pass:
- Median within 20% of NJ statewide median ($560,000)
- County-level medians within 25% of known 2024-2025 values
- All zip codes are 5-character strings (no leading-zero loss)
